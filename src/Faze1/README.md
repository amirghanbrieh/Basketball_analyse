## 🎯 نمای کلی

این اسکریپر داده‌های NBA را از وب‌سایت Basketball-Reference.com برای فصل‌های **۲۰۱۸ تا ۲۰۲۶** (۹ فصل) استخراج می‌کند. اطلاعات جمع‌آوری شده شامل موارد زیر است:

- جوایز و رهبران آماری هر فصل (قهرمان، MVP، بهترین تازه‌کار، رهبران امتیاز، ریباند، پاس منجر به گل و بردهای تیمی)
- لیست کامل بازیکنان فعال با جزئیات حرفه‌ای
- خلاصه اطلاعات تیم‌ها و تاریخچه آن‌ها
- داده‌های کامل هر تیم در هر فصل (رکورد، مربی، رتبه‌بندی‌ها، تعداد تماشاگران، نتایج پلی‌آف)
- آمار هر بازیکن در هر فصل به همراه رتبه‌بندی در شاخص‌های مختلف
- امکان پردازش فایل‌های HTML ذخیره‌شده محلی برای استخراج اطلاعات بیوگرافی بازیکنان

تمام خروجی‌ها به صورت فایل‌های CSV در پوشه `data/` ذخیره می‌شوند و بازخورد اجرا با رنگ‌های زیبا و کتابخانه `colorama` نمایش داده می‌شود.

## 🚀 روش اجرا

برای اجرای کامل اسکریپت و استخراج تمام داده‌ها، کافی است دستور زیر را اجرا کنید:

```bash
python main.py
```

### ترتیب مراحل اجرا

۱. **استخراج اطلاعات فصلی** (قهرمان، MVP، رهبران آماری)  
۲. **استخراج لیست بازیکنان فعال**  
۳. **استخراج خلاصه تیم‌ها و لینک‌ها**  
۴. **استخراج داده‌های کامل تیمی-فصلی** (جزئیات هر تیم در هر فصل)  
۵. **استخراج آمار بازیکنان و تیم‌ها با رتبه‌بندی**  
۶. **(اختیاری) پردازش فایل‌های HTML محلی بازیکنان** (در صورت وجود پوشه `basketball_pages/gamers`)


### تنظیمات اولیه در `main.py`

شما می‌توانید بازه فصل‌ها و مسیر پوشه خروجی را در بخش `main()` تغییر دهید:

```python
START_YEAR = 2018      # سال شروع
END_YEAR = 2026        # سال پایان
OUTPUT_DIR = 'data'    # پوشه خروجی
PLAYER_HTML_DIR = 'basketball_pages/gamers'  # مسیر فایل‌های HTML محلی (اختیاری)
```

## 📊 طرح داده

### `seasons.csv`

| ستون | توضیح |
|-------|-------|
| `Season` | فصل به صورت `سال شروع-سال پایان` |
| `Champion` | نام تیم قهرمان |
| `MVP` | نام باارزش‌ترین بازیکن |
| `ROY` | بهترین تازه‌کار |
| `PPG_Leader` | رهبر امتیاز |
| `RPG_Leader` | رهبر ریباند |
| `APG_Leader` | رهبر پاس منجر به گل |
| `WS_Leader` | رهبر بردهای تیمی |

### `nba_active_players.csv`

| ستون | توضیح |
|-------|-------|
| `PlayerID` | شناسه یکتا |
| `PlayerName` | نام کامل |
| `PlayerURL` | لینک صفحه |
| `From` | سال شروع حضور |
| `To` | سال پایان حضور |
| `Pos` | پست |
| `Ht` | قد |
| `Wt` | وزن |
| `BirthDate` | تاریخ تولد |
| `College` | دانشگاه |

### `NBA_2018_2026.csv` (داده‌های تیمی-فصلی)

این فایل شامل ستون‌های زیر است:

- `Season`، `Conference`، `Team`
- `League Champion`، `MVP`، `ROY`، `PPG Leader`، `RPG Leader`، `APG Leader`، `WS Leader`
- `Conference Wins`، `Conference Losses`، `Conference W/L%`، `GB`، `PS/G`، `PA/G`، `Conference SRS`
- `Record`، `Coach`، `Executive`، `Team PTS/G`، `Opponent PTS/G`، `SRS`، `Pace`
- `Offensive Rating`، `Defensive Rating`، `Net Rating`، `Expected W-L`
- `Preseason Odds`، `Arena`، `Attendance`، `Playoffs`

### `nba_players_2018_2026.csv`

شامل تمام ستون‌های جدول `totals_stats` به علاوه:

- `Season` و `Season_Year`
- رتبه‌بندی برای هر شاخص (مانند `PTS_Rank`، `TRB_Rank` و ...)
- `Combined_Score` و `Overall_Rank` (رتبه کلی ترکیبی بر اساس امتیاز، ریباند و پاس)

## ⚙️ جزئیات فنی

- **زبان**: Python 3
- **کتابخانه‌های اصلی**: 
  - `requests` برای درخواست‌های HTTP
  - `BeautifulSoup4` برای تجزیه HTML
  - `pandas` برای پردازش و ذخیره داده‌ها
  - `colorama` برای خروجی رنگی
- **معماری**: شی‌گرا با کلاس‌های مجزا برای هر بخش
- **مدیریت تأخیر**: استفاده از `time.sleep` برای جلوگیری از مسدود شدن IP
- **مدیریت خطا**: استفاده از `try/except` برای پایداری در برابر خطاهای شبکه و ساختار صفحات
- **پردازش موازی**: خیر – به‌صورت ترتیبی برای جلوگیری از بار زیاد روی سرور
- **پشتیبانی از فایل‌های محلی**: کلاس `PlayerPageParser` قابلیت پردازش صفحات ذخیره‌شده را دارد
